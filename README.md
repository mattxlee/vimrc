## 安装

### Neovim

首先需要安装neovim，可以访问neovim的官网网站[neovim.io](https://neovim.io)来获得neovim的详细安装步骤，如果是使用MacOS，那么简单的使用brew来安装就可以了。

```bash
brew install neovim
```

你还需要有安装python，并且pip命令是可用的，若没有安装好python，请查阅python官网的安装说明，若是MacOS，可以使用brew来安装python。

```bash
brew install python2
```

安装好了neovim后，就可以使用库中的`install-neovim-conf.sh`来进行配置和插件的安装。直接执行以下命令：

```bash
./install-neovim-conf.sh
```

若你需要删除所有的neovim相关配置，可以使用`--clear`参数，然后再执行`install-neovim-conf.sh`则可以全新安装所有的配置。

```bash
./install-neovim-conf.sh --clear
```

执行完成脚本后，neovim的基础配置已经完成，这时neovim已经是可以正常使用。

### 插件说明以及配置

#### 配色

最喜欢的配色是gruvbox，所以我在vim里默认也只会安装这一个配色插件，其实这个插件有白天夜晚的配置，修改`set background=dark`或`set background=light`可以改变这个配置，默认是使用夜晚的配色。

#### 彩虹括号

这个是受到了vscode的彩虹括号的启发，找了一下果然vim里也有这个插件，安装完后发现彩虹括号会导致cmake文件的显示有些问题，所以添加了额外的配置，让彩虹括号在cmake文件中关闭了。

#### 额外配置

使用`editorconfig-vim`以及`vim-localvimrc`这两个可附加的额外配置插件。

* `editorconfig`是各种编辑器都可以使用的通用配置，可以配置当前的工作区的一些编辑设置，比如tab多少空格等。

* `localvimrc`是可以在本地创建一个`.lvimrc`文件，这样可以针对本地的工程添加一些特殊的配置，我一般会用它来配置当前的编辑区ruler.


#### 全局搜索

当前目录中的文件全局搜索是挺重要的一个功能，这里使用`ag`和`ctrlsf`来完成这项工作，在使用它之前，要安装相应的命令行工具，可以直接执行安装脚本：

```base
./install-global-search.sh
```

#### ctag

ctag需要额外的安装，同时还有一个配置文件`.ctags`需要扔到`$HOME`目录下，直接执行脚本`./install-ctags.sh`来安装。

#### 快速文件跳转

快速文件跳转是一系列的插件，跳转文件是写代码中比较重要的一个操作。例如，需要搜索文件名跳转，或者c/c++中头文件、源文件切换，或者查找某些定义等。

1. `vim-fswitch`是我改过的一个c/c++头源文件跳转的插件，当时应该是有一些习惯上的小问题，所以我克隆了原作者的库进行了一些小调整，使用原作者的应该也是可以的。这个插件使用`ctrl+h`这个快捷键。

1. `nerdtree`是个老牌的文件树插件了，同时还安装了git的支持，这样在插件里可以看到当前的工作目录中的工作文件的git修改状况。在非插入状态下使用`ctrl+j`这个快捷键可以在当前的窗口中打开文件树。

1. `LeaderF`是个非常棒的tag或文件跳转插件，我使用它来代替了`ctrlp`插件，并且同时可以对符号和当前文件的函数进行快速跳转。使用`<leader>f`可以查看当前文件中的所有的函数列表，并输入关键字可以快速跳转，使用`<leader>g`可以查看当前项目中所有的已经分析出来的关键词，并且输入关键字可以快速跳转。


#### 快速编辑

这一堆的插件都是一些挺不错的辅助增强插件，比如删除多余空格，快速注释，快速文件内跳转以及快速对齐等。

#### git支持

对于git的支持当然非常重要，不得不说`vim-fugitive`这个插件真是越来越好用，比起vscode来说真是强出不少，这也是导致我一直在vscode不适应的一个很重要的原因。

另外，我把.gitconfig文件加到项目中了，可以执行脚本`./install-git-conf.sh`来把.gitconfig文件拷贝到`$HOME`目录下。

#### 常用语言语法高亮的支持

#### 状态栏

使用的是老牌的`airline`，这个插件使用下来也没有什么问题。

#### 字体

字体其实和vim配置没有太多的关系，但是我一同写到了这个项目里，执行`./install-nerd-fonts.sh`这个脚本会使用brew来安装我平常使用的Hack字体。
